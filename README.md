## 📱 앱 소개

TodaysPlayer는 **스포츠 매칭 전문 플랫폼**입니다. 카카오톡, 밴드, 전화 등에 흩어져 있는 불편한 일정 관리와 모호한 모집 현황을 해결하는 통합 솔루션을 제공합니다. 

**게임 일정 관리 → 용병 모집 참여/투표 → 알림 리마인더**까지 전체 과정을 자동화하여 스포츠 커뮤니티를 효율적으로 연결합니다.

---

## ✨ 주요 기능

### 🏠 **스마트 홈 대시보드**
- **다음 경기 카드**: 예정된 경기 정보와 길찾기
- **주변 매치**: 근처 풋살/축구 매치 실시간 목록

### ⚽ **매치 관리 시스템**
- **매치 생성**: 팀장/관리자가 매치 일정 등록
- **실시간 모집 현황**: 참가 신청 현황 실시간 확인
- **참가자 프로필 검토**: 경력, 포지션, 최근 참여 횟수 등 확인
- **승인/거부 시스템**: 수락/거부 버튼으로 참가자 관리

### 📝 **매치 신청 프로세스**
- **날짜 선택**: 참여 가능한 날짜 선택
- **신청 정보 입력**: 포지션, 실력 수준 등 상세 정보 입력
- **신청 완료**: "신청 접수됨. 관리자 승인 대기 중" 메시지 표시

### 📊 **개인 통계 및 관리**
- **활동 내역**: 과거 경기 참여 기록 및 상세 조회
- **내 매치 목록**: 신청/참여한 매치 목록 관리
- **매치 수정/취소**: 개설한 매치 정보 수정 및 취소
- **참가자 관리**: 개설 매치의 참가자 목록 확인 및 관리
- **신뢰도 시스템**: 매너 및 참여율 기반 신뢰도 점수

### 🔐 **사용자 인증**
- **로그인**: 이메일 로그인
- **계정 복구**: 아이디/비밀번호 찾기 기능

## 👥 사용자 시나리오

### 🏆 **모집자(팀장/관리자) 시나리오**
1. **매치 등록**: "토요일 7시, 송파풋살장, 3명 모집" 공고 등록
2. **실시간 확인**: 앱에서 신청 현황 실시간 확인
3. **참가자 검토**: 신청자 프로필(경력, 포지션, 최근 참여 횟수) 확인
4. **승인/거부**: 수락/거부 버튼으로 참가자 관리

### ⚽ **용병(참여자) 시나리오**
1. **매치 검색**: "토요일 저녁 풋살" 검색하여 매치 발견
2. **신청**: 참여 신청 버튼 클릭
3. **대기**: "신청 접수됨. 관리자 승인 대기 중" 메시지 확인
